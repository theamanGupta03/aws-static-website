What powers this site

☁️

Amazon S3

All HTML, CSS, and image files are stored in a private S3 bucket. No public access — only CloudFront can read them.

⚡

CloudFront CDN

Files are cached at 400+ global edge locations. Visitors get fast load times no matter where they are in the world.

🔌

API Gateway

Dynamic API calls are routed through CloudFront to API Gateway, keeping the frontend and backend under one domain.

🏗️

Terraform IaC

Every AWS resource above is defined as code in .tf files. One command deploys the entire infrastructure.
