About This Project

A hands-on cloud practice project — from zero to deployed.

Why I built this

This project is a practice demo to learn how to host static websites on AWS using industry-standard tools. Instead of just reading about cloud infrastructure, I built it myself — deploying real AWS services with Terraform.

What I learned

  • ✅ How S3 stores static files privately
  • ✅ How CloudFront distributes content globally
  • ✅ How API Gateway handles dynamic requests
  • ✅ How Terraform creates infrastructure as code
  • ✅ How IAM policies secure access between services

Tech Stack

  • HTML Structure
  • CSS Styling
  • JS Interactivity
  • S3 File storage
  • CloudFront CDN
  • API Gateway API routing
  • Terraform Infrastructure
